## Key Points to Remember

### 1. Java Fix
```javascript
// WRONG: File was Main_123456.java but class was Main
// Result: ClassNotFoundException

// RIGHT: File is always Main.java and class is Main
// Result: Works perfectly!
```

### 2. C Fix
```javascript
// WRONG: Always looked for .exe (Windows-only)
// On Linux: gcc creates no .exe â†’ FileNotFound

// RIGHT: Check os.platform() and use correct extension
// Windows: .exe âœ…  Linux: no extension âœ…
```

### 3. Process Execution
```javascript
// WRONG: execSync(`java -cp "${tempDir}" Main_123456`)
// Issues: Shell parsing, space handling, command mixing

// RIGHT: spawnSync('java', ['-cp', tempDir, 'Main'])
// Benefits: Direct execution, no shell parsing
```

### 4. Response Format
```javascript
// WRONG: Confusing fields (compileOutput sometimes)
response = {
  stdout: '...',
  stderr: '...',
  compileOutput: '...'  // Unclear
}

// RIGHT: Clear, structured response
response = {
  success: true/false,
  output: '...',
  error: '...',
  compileError: true/false  // Clear!
}
```
